métodos jQuery

Com jQuery é possível animar os elementos HTML, fazendo-os esconderem, aparecerem, de acordo com cada evento que ocorrerá na página

.show()

O método .show() permite mostrar elementos HTML, de acordo com o evento indicado

.hide()

O método .hide() permite esconder elementos HTML, de acordo com o evento indicado

.toggle()

O método .toggle() permite alterar a visibilidade dos elementos HTML - a junção do .show() e .hide()

.slideUp()

O método .slideUp() permite fazer drillup em elementos HTML, de acordo com o evento indicado

.slideDown()

O método .slideDown() permite fazer drillDown em elementos HTML, de acordo com o evento indicado

.slideToggle()

O método .slideToggle() permite alterar a visibilidade dos elementos HTML - junção do .slideUp() e .slideDown()

.fadeIn()

O método .fadeIn() permite mostrar suavemente elementos HTML, de acordo com o evento indicado

.fadeOut()

O método .fadeOut() permite esconder suavemente elementos HTML, de acordo com o evento indicado

.fadeTo()

O método .fadeTo() permite alterar a visibilidade dos elementos HTML, indicando o nível de opacidade

.animate()

O método .animate() permite trocar o CSS do elemento selecionado, gerando animações

.css()

Através do método .css() é possível definir formatações CSS ao elemento selecionado

.html()

Através do método .html() é possível alterar o conteúdo de elementos HTML

.before()

Com o método .before() é possível adicionar conteúdos antes do elemento HTML selecionado

.after()

Com o método .after() é possível adicionar conteúdos depois do elemento HTML selecionado

.height()

Com o método .height() é possível alterar a altura do elemento HTML

.width()

Com o método .width() é possível alterar a largura do elemento HTML
