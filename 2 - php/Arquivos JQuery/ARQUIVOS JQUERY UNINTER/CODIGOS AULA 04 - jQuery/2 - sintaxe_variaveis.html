sintaxe jQuery

A sintaxe jQuery é orientada para selecionar elementos HTML e efetuar ações. A sintaxe básica jQuery é:


  • $: definição que será utilizado um elemento jQuery
  • (seletorHTML): elemento que será animado - elemento, id ou tag
  • .ação(): método que será chamado

variáveis

Assim como em JavaScript puro, em jQuery o padrão de declaração de variáveis é:

var x = "informação"

uma variável poderá armazenar strings (texto), integers (números), arrays (vetores), entre outros datatypes
